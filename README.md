## 1. 상세 내용
COME BACK HOME 게임은 인공지능이 보편화된 미래사회를 배경으로 한다. AI, 즉 플레이어가 깨어나며 게임이 시작되고 외부의 공격으로 인해 에러가 난 자신을 발견한다. 자동으로 복구하기 위해 리부팅을 시작하며, 그 과정을 미로찾기로 표현하였다. 
미로찾기를 하며 다양한 방해요소를 마주치게 되고, 이 요소들은 판뒤집기, 조작키 전환 같은 기능으로 구현된다.
